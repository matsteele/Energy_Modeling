
# starting the project up 

1) cd into server
```bash
cd server
```
2) start virtual environment
```bash
source venv/bin/activate
```
3) install dependencies 
```bash
 pip3 install -r requirements.txt
```
4) start server
```bash
python main.py
```

5) start front-end
cd into static
webpack --watch
also start Sass styling
sass --watch style/style.scss:style/style.css
